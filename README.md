# Algorithm-With-C
#### 使用环境

* Ubuntu18.04 LTS
* g++ 7.3.0
* CMake 3.10.2
* Vim 8.0.1453
* C++11

#### 常用数据结构

* Queue.h 队列模板类
* Stack.h 栈模板类
* Array.h
* ArrayStack.h 
* ArrayQueue.h
* LoopQueue.h

#### 如何使用代码

* 新建build文件夹 `mkdir build`
* 进入build文件夹 `cd build`
* cmake配置 `cmake ..`
* make 编译 `make`
* 执行代码 `./DataStructure`
